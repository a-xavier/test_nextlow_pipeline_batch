## Read Me

We do one sub workflow for each possible input.
- Paired-end FastQ
- Single FastQ
- BAM/CRAM/SAM (unaligned reads)
- BAM/CRAM/SAM (aligned reads)
- BAM/CRAM/SAM (unaligned with methylation)
- BAM/CRAM/SAM (aligned with methylation)
- Fast5 (Nanopore)
- Pod5 (Nanopore)
- VCF (BCF / VCF.GZ)
